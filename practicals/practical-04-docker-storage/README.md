\# 🚀 Practical 04: Docker Storage (Volumes)



\## 📌 Objective

To understand Docker volumes and data persistence.



\---



\## 🛠️ Commands Used

docker volume create my-volume  

docker run -it -v my-volume:/data ubuntu bash  



\---



\## 📝 Steps

1\. Created Docker volume  

2\. Mounted volume to container  

3\. Created file inside container  

4\. Verified data persistence
